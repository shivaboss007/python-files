# 🐍 Python Exam Practice Project

## 📌 Project Overview

This project consists of multiple Python programming problems designed to test and improve problem-solving skills. Each problem focuses on core Python concepts such as loops, conditionals, lists, and string manipulation.

---

## 🎯 Objectives

* Strengthen Python fundamentals
* Practice logic building for coding exams
* Improve problem-solving and algorithmic thinking

---

## 🛠️ Technologies Used

* Python 3
* Jupyter Notebook

---

## 📂 Problems Covered

### 🔹 1. Reverse a String

Reverses a given string using slicing.

### 🔹 2. Filter Even Numbers

Extracts even numbers from a list using list comprehension.

### 🔹 3. Count Vowels

Counts total vowels in a given string.

### 🔹 4. Middle Element(s) of List

Finds the middle element(s) depending on list length (odd/even).

### 🔹 5. Alternating Sign List

Converts list elements into alternating positive and negative values.

### 🔹 6. Number Ladder Pattern

Prints numbers in a ladder format using nested loops.

### 🔹 7. Password Strength Checker

Classifies password as Weak, Moderate, or Strong based on conditions.

### 🔹 8. Spy Number

Checks if sum of digits equals product of digits.

### 🔹 9. Alien Language Translator

Transforms a word by replacing alternate characters with `*`.

### 🔹 10. Magic Multiples

Prints:

* "Boom" for multiples of 4
* "Bang" for multiples of 6
* "BoomBang" for multiples of both
